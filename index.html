Flexbox Row

Use these three properties to create a flexbox row

.row {
    display:flex;
    flex-direction:row;
    flex-wrap:wrap;
}

Flexbox Column

Use this to create a Flexbox column layout

.column {
    display:flex;
    flex-direction:row;
}
